# How to change compile configuration
## 1. Enable DEBUG (Important!)
# What should we do to make changes take effect

1. Rebuild compiler

make -f Makefile-parallel clean
make -f Makefile-parallel

# 使用4个并行任务（通常为核心数）
make -f Makefile-parallel -j4

# 或者让make自动决定最佳并行数（推荐）
make -f Makefile-parallel -j$(nproc)

2. Rebuild Libs
./tests/compileLib.sh

3. Verify Libs
ls -l build/lib/sylib++.o
nm build/lib/sylib++.a | grep getint

4. Run tests
./test -r          # clean cache
./test functional  # baseline
./test functional passesTest.sh # our custom compiler
